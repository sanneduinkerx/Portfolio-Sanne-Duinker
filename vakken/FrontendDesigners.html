Top Rated Movielist - JSON file

Frontend voor Designers

Maart 2020

Voor deze opdracht wilde ik me meer verdiepen in de frontend kant van ontwerpen. Ik vind het fascinerend om vanuit niets iets te bouwen en vond dit een perfecte kans. Door middel van Design Princicples en coderen heb ik een website gebouwt, Top Rated Movies. Ik heb hierbij geleerd te werken met een JSON file en hoe je dat allemaal via javascript doet. Daarnaast weet ik nu ook hoe ik verschillende states, loading en error state, kan toepassen binnen coderen. Bekijk hier de website.
